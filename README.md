# ingenieriadeprompts

## AulaIA: Laboratorio de Prompts

Este repositorio contiene la interfaz principal de un portafolio web interactivo diseñado para docentes de educación media de Bogotá. El proyecto ofrece un espacio modular con actividades formativas sobre prompts, producción hipermedia, generación visual, validación académica y chatbots educativos.

### ¿Qué incluye?

- `index.html`: Página principal con diseño semántico y Bootstrap 5.
- `styles.css`: Estilos personalizados con paleta sobria y tipografía legible.
- `script.js`: Lógica interactiva para búsqueda de módulos y apertura de modales.

### Estructura del portafolio

El sistema presenta 8 módulos de aprendizaje:

1. Módulo 1: Gema – Guía metodológica para la estructura de un prompt.
2. Módulo 2: Producción Hipermedia con NotebookLM / Google Vids.
3. Módulo 3: Producción Hipermedia con NoteGPT.
4. Módulo 4: Producción Hipermedia Visual (Perplexity, ChatGPT o Gemini).
5. Módulo 5: Consensus – Buscador y validador de artículos científicos.
6. Módulo 6: Mizou / Botpress – Chatbot tutor interactivo.
7. Módulo 7: Herramientas Visuales I – Infografías educativas con IA.
8. Módulo 8: Herramientas Visuales II – Mapas conceptuales y diagramas didácticos.

### Créditos institucionales

Proyecto pensado para la Maestría en Educación en Tecnología de la Universidad Distrital Francisco José de Caldas. Todos los recursos visuales y ejemplos están diseñados como contenido simulador con licencias Creative Commons.

### Uso

Abrir `index.html` en el navegador o servirlo desde un servidor local para ver la interfaz y probar la navegación interactiva.
